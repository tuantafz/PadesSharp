# License Provenance

## Summary

PadesSharp is a clean-room implementation of a modern PDF signing library built on top of the iTextSharp 4.1.6 LGPL fork.

## Base Library: iTextSharp 4.1.6

- **License**: LGPL 2.1
- **Source**: https://github.com/schourode/iTextSharp-LGPL
- **NuGet**: https://www.nuget.org/packages/iTextSharp-LGPL/4.1.6
- **Original project**: https://sourceforge.net/projects/itextsharp/

The `LegacyPdfCore` project contains or references iTextSharp 4.1.6 source code, which is licensed under LGPL 2.1.
This project complies with LGPL by:
- Keeping the LGPL core as a separate assembly (`LegacyPdfCore`)
- Not modifying the LGPL core unless strictly necessary for API compatibility
- Documenting all modifications to the LGPL core in this file

## New Modules (Original Implementation)

The following modules are **original implementations** by PadesSharp contributors,
written from scratch based on public standards. They do **NOT** copy, derive from, or port
code from iText 5, iText 7, or any AGPL/commercial source. All modules are distributed
under LGPL 2.1+.

| Module | Standards Used |
|--------|---------------|
| `ModernPdf.Abstractions` | N/A (interfaces only) |
| `ModernPdf.Crypto` | RFC 5652 (CMS), RFC 3161 (TSA), RFC 6960 (OCSP), RFC 5280 (X.509) |
| `ModernPdf.Signing` | ISO 32000-1 (PDF), ISO 32000-2 (PDF 2.0) |
| `ModernPdf.Pades` | ETSI EN 319 102-1 (PAdES), ETSI TS 102 778 (PAdES baseline) |
| `ModernPdf.Pkcs11` | PKCS#11 v2.40 (Cryptoki) |
| `ModernPdf.Validation` | ETSI EN 319 102-2, ISO 32000 |
| `ModernPdf.Appearance` | ISO 32000-1 §12.5.6.8, PDF Visual Signature |
